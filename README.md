# Guard_Dog

Lightweight face-monitoring utility that logs activity and captures intruder screenshots.

## Quickstart

1. Create and activate a Python virtual environment.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
python main.py
```

## Project layout

- `core/` - core detection and capture logic
- `ui/` - user interface modules
- `data/` - captured faces, intruders, and activity log
